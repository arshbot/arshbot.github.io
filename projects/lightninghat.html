What

A raspberry pi connected to an ardiuno controlling some motors, controlled by a django server running on top of an lnd node. All held together by a yellow helmet

Why

Built for lnconf 2019, I thought it'd be cool if I built a demo showing how easy it would be to accept bitcoin over lightning network in germany, a country well known for it's piss poor connectivity. Heavily inspired by a tweet of a man who built something similar except the fans spun every time a github pr was merged. And also because I wanted to play with an arduino :)
